# Contributing to Learning Convex Decomposition via Feature Fields

Thanks for your interest in contributing to this project. This document
explains how to submit a contribution.

## Signing Your Work

We require that all contributors "sign off" on their commits. This sign-off
certifies that you wrote the contribution, or otherwise have the right to
submit it under the same open source license used by this project.

Any pull request that contains commits which are not signed off will not be
accepted.

To sign off on a commit, use the `--signoff` (or `-s`) flag when committing:

    $ git commit -s -m "Add cool feature."

This appends a `Signed-off-by` trailer to your commit message:

    Signed-off-by: Your Name <your.email@example.com>

Make sure your name and email are configured in git first:

    $ git config --global user.name "Your Name"
    $ git config --global user.email "your.email@example.com"

If you forgot to sign off on a commit you have already made, you can amend
it:

    $ git commit --amend --signoff
